## 📋 PROPÓSITO

Benchmark avançado para testar modelos Ollama locais com métricas detalhadas:
- Tempo de resposta
- Tokens por segundo
- Uso de memória RAM
- Qualidade das respostas

---

## 🚀 INSTALAÇÃO RÁPIDA

```bash
# Clone o repositório
git clone https://github.com/sabrina-ia/ollama-benchmark-beta.git
cd ollama-benchmark-beta

# Execute o script
chmod +x ollama-benchmark-beta-v0.0.1.sh
./ollama-benchmark-beta-v0.0.1.sh
```
